# Sokol Skuhrov Website

Static website for Sokol Skuhrov built with Next.js, Markdown, and Sveltia CMS.

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Content

- Pages: `content/pages/*.md`
- Posts: `content/posts/*.md`
- Images: `public/images/`

## Admin

Edit content via `/admin` (Sveltia CMS with GitHub OAuth).

## Deploy

Automatically deployed to Vercel on push to `main`.

See [DEPLOYMENT.md](./DEPLOYMENT.md) for detailed deployment infrastructure documentation.
