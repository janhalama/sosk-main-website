# Content Directory

This folder contains Markdown content used to statically generate the site.

- `pages/`: Markdown files for static pages (e.g., Domů, Činnost, Fotogalerie, Sponzoři, Kontakty).  
- `posts/`: Markdown files for news posts (Akce) with frontmatter.

Frontmatter (posts):
- `title` (string), `date` (ISO), `image` (optional path under `public/`), `summary` (optional), `slug` (string).
